2. Ranking de Tecnologias

  1. WebAssembly (Wasm)

    O WebAssembly é uma tecnologia revolucionária que permite a execução de linguagens de baixo nível (como C, C++ e Rust) diretamente no navegador com desempenho quase nativo. Ele não substitui as tecnologias web tradicionais, mas atua em conjunto com elas para viabilizar a execução de aplicações extremamente pesadas e complexas no browser, como editores de vídeo, jogos 3D avançados e softwares de modelagem, que antes eram exclusivos de programas instalados no computador.

  2. Node.js

    o Node.js é um ambiente de execução que permitiu que o JavaScript saísse dos navegadores e fosse utilizado também na construção de servidores (back-end). Com uma arquitetura assíncrona e orientada a eventos, ele é altamente leve e escalável, sendo a espinha dorsal de inúmeras APIs modernas, serviços de streaming e aplicações em tempo real ao redor do mundo.

  3. React.js

    O React permite que desenvolvedores construam aplicações web de página única (SPAs) que são extremamente rápidas, modulares e fáceis de manter, consolidando-se como o padrão absoluto da indústria para interfaces interativas.

  4. HTML5 e CSS3

    A base estrutural e visual de absolutamente tudo o que existe na internet. O HTML5 fornece a semântica e a marcação de todo o conteúdo, permitindo a integração nativa de áudios, vídeos e gráficos, enquanto o CSS3 dita a estilização, a responsividade para dispositivos móveis e as animações.

  5. JavaScript

    JavaScript é a linguagem de programação nativa da internet. É ele o responsável por dar vida às páginas, transformando documentos estáticos em aplicações dinâmicas que processam dados e reagem em tempo real aos comandos do usuário.

3. Glossário Técnico

Frontend
Client-side
Camada de desenvolvimento responsável pela interface com a qual o usuário interage diretamente no navegador.
Host
Qualquer computador, servidor ou dispositivo final conectado a uma rede de computadores que possui um endereço de IP atribuído e pode trocar dados com outras máquinas.
Forma abreviada de "Web Host" (Serviço de Hospedagem). Refere-se a uma empresa comercial que aluga espaço em seus servidores de alta disponibilidade para armazenar os arquivos de um site, garantindo que ele fique acessível na internet.
API
Um conjunto de rotinas, regras e protocolos que atua como um mensageiro, permitindo que dois softwares ou sistemas diferentes se comuniquem e troquem dados de forma segura.
DOM
Uma representação hierárquica e estruturada em forma de árvore do documento HTML carregado no navegador. O DOM atua como uma interface de programação que permite ao JavaScript encontrar, acessar e modificar os elementos, a estrutura e os estilos da página em tempo real, sem precisar recarregá-la
Tag
No desenvolvimento HTML, é um comando estrutural delimitado por colchetes angulares (como p para parágrafo ou img para imagem) que instrui o navegador sobre como exibir um conteúdo específico na página.
Uma etiqueta ou palavra-chave descritiva anexada a um post de blog, vídeo ou produto digital.
DNS
Sistema de nomes de domínio; traduz nomes de domínio legíveis por humanos em endereços IP numéricos.
HTTP
Protocolo de transferência de hipertexto; define as regras de comunicação entre clientes e servidores na Web.
